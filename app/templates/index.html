Quick Start

  1. Connect your Gmail account
  2. Create a new campaign
  3. Upload your leads CSV
  4. Review inferred schema and sample emails
  5. Approve and run (or dry-run first)
✨

Features

  • AI-powered CSV schema inference
  • Personalized email generation
  • Campaign-level approval workflow
  • Row-level review for risky emails
  • Dry-run mode (default)
  • Idempotency to prevent duplicates
  • Full audit logging
  • Resumable workflows

💡 Getting Started Tip

Start with a small CSV file (10-20 rows) to test your campaign. Use dry-run mode to preview emails before sending. The AI will automatically detect email columns and personalization fields from your data.
